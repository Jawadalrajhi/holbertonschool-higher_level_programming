# Python – More Data Structures: Sets & Dictionaries

This project dives deeper into Python’s built-in data structures, focusing on how sets and dictionaries work and how to manipulate them efficiently. Throughout the exercises, I also practiced using functional tools like `lambda`, `map`, `filter`, and `reduce`.

## Tests :heavy_check_mark:

* All test files are available in the **[tests](./tests)** directory (provided by Holberton School).

## Tasks :page_with_curl:

### **0. Squared simple**
* **File:** [`0-square_matrix_simple.py`](./0-square_matrix_simple.py)  
* Returns a new matrix where each integer is squared.  
* The original matrix is left unchanged.  
* No module imports.

---

### **1. Search and replace**
* **File:** [`1-search_replace.py`](./1-search_replace.py)  
* Creates a new list where all occurrences of a specific element are replaced with another value.  
* No imports.

---

### **2. Unique addition**
* **File:** [`2-uniq_add.py`](./2-uniq_add.py)  
* Adds all unique integers in a list and returns the sum.  
* No imports.

---

### **3. Present in both**
* **File:** [`3-common_elements.py`](./3-common_elements.py)  
* Returns a set containing elements shared between two sets.  
* No imports.

---

### **4. Only differents**
* **File:** [`4-only_diff_elements.py`](./4-only_diff_elements.py)  
* Returns elements that appear in only one of the two sets.  
* No imports.

---

### **5. Number of keys**
* **File:** [`5-number_keys.py`](./5-number_keys.py)  
* Returns the total number of keys in a dictionary.  
* No imports.

---

### **6. Print sorted dictionary**
* **File:** [`6-print_sorted_dictionary.py`](./6-print_sorted_dictionary.py)  
* Prints a dictionary sorted alphabetically by key.  
* Keys are assumed to be strings.  
* No imports.

---

### **7. Update dictionary**
* **File:** [`7-update_dictionary.py`](./7-update_dictionary.py)  
* Adds or updates a key/value pair in a dictionary.  
* No imports.

---

### **8. Simple delete by key**
* **File:** [`8-simple_delete.py`](./8-simple_delete.py)  
* Deletes a key from a dictionary if it exists.  
* No imports.

---

### **9. Multiply by 2**
* **File:** [`9-multiply_by_2.py`](./9-multiply_by_2.py)  
* Returns a new dictionary with all integer values doubled.  
* No imports.

---

### **10. Best score**
* **File:** [`10-best_score.py`](./10-best_score.py)  
* Returns the key associated with the highest integer value in a dictionary.  
* Returns `None` if the dictionary is empty or no valid score exists.
